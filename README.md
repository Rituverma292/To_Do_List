To-Do List Web Application
A modern, responsive To-Do List web application built using HTML, CSS, and JavaScript to help users manage and track daily tasks efficiently.
Features:
1.Add new tasks
2.Edit existing tasks
3.Delete tasks
4.Mark tasks as completed
5.Add due dates
6.Filter tasks (All / Completed / Pending)
7.Data stored using Local Storage
8.Fully responsive design
